3. Churn Evaluation / முடிவுகளைக் காணுதல்

Step 3
English

Click the Simulate Churn Risk button. The Python Machine Learning model instantly returns:

  • Churn Probability % (e.g. 85% risk)
  • Risk Drivers (e.g. month-to-month contracts)
  • Recommendations for customer retention
தமிழ்

**Simulate Churn Risk** பொத்தானை அழுத்தியவுடன், Python ML மாடல் வாடிக்கையாளரின் வெளியேறும் அபாயத்தைக் கணக்கிட்டுக் காட்டும்:

  • வெளியேற்ற சதவீதம் % (எ.கா. 85% அபாயம்)
  • முக்கியக் காரணங்கள் (எ.கா. மாதாந்திர ஒப்பந்தங்கள்)
  • வாடிக்கையாளரைத் தக்கவைக்க **ஆலோசனைகள்**
